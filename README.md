# MNIST/CIFAR10 ML Project
Project for CSCI 4521 Applied Machine Learning, implementing various models and methods to classify digits from the MNIST dataset and using a custom CNN architecture based off ResNet to classify images from the CIFAR-10 dataset.

Project code can be viewed in Colab here: [MNIST_CIFAR_ML_project.ipynb](https://github.com/Viper4/MNIST-CIFAR10-ML-project/blob/main/MNIST_CIFAR_ML_project.ipynb)

Final project report detailing model architecture, methodology, and results can be viewed here: [MNIST_CIFAR ML Project Report.pdf](https://github.com/Viper4/MNIST-CIFAR10-ML-project/blob/main/MNIST_CIFAR%20ML%20Project%20Report.pdf)
